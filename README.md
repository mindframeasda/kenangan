# 🎓 Website Kenangan Kelas (XII AKL 1 / XII AKUNTANSI 1)

Website kenangan premium yang dirancang untuk merayakan perjalanan dan persahabatan kelas. Dibangun dengan **React**, **Framer Motion**, dan **Vite** untuk performa maksimal dan animasi yang halus.

## ✨ Fitur
- **Hero Section**: Tampilan pembuka yang elegan dengan latar belakang cinematic.
- **Statistik Kelas**: Ringkasan jumlah siswa dan momen yang telah dilewati.
- **Galeri Momen**: Grid foto interaktif dengan efek hover yang premium.
- **Desain Glassmorphism**: Estetika modern dengan transparansi dan blur.
- **Full Responsive**: Tampilan optimal di HP, Tablet, dan Laptop.

## 🚀 Cara Menjalankan

### Lokal (Development)
1. Pastikan Anda memiliki Node.js terinstal.
2. Jalankan perintah:
   ```bash
   npm install
   npm run dev
   ```

### Deployment (Vercel)
Website ini sudah siap untuk di-deploy ke Vercel:
1. Hubungkan repository ini ke akun Vercel Anda.
2. Vercel akan secara otomatis mendeteksi pengaturan Vite.
3. Klik **Deploy**.

## 🎨 Teknologi
- **Core**: React.js
- **Styling**: Vanilla CSS (Premium Design System)
- **Animation**: Framer Motion
- **Icons**: Lucide React
- **Build Tool**: Vite
