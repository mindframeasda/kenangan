# 🎓 Struktur Website Kenangan Kelas (Premium)

Struktur website ini dirancang khusus untuk kebutuhan website kenangan kelas yang modern, estetik, dan mudah di-deploy ke Vercel.

## 📁 Struktur Folder
```text
WEB Kelas/
├── public/                 # Aset statis (Gambar, Ikon)
│   └── assets/             # Gambar hero dan placeholder
├── src/
│   ├── App.jsx             # Komponen utama & Layout
│   ├── App.css             # Styling komponen (Premium Design)
│   ├── index.css           # Design tokens & Global styles
│   └── main.jsx            # Entry point React
├── index.html              # Metadata & SEO setup
├── vercel.json             # Konfigurasi deploy Vercel
├── package.json            # Dependensi (React, Framer Motion, Lucide)
└── README.md               # Panduan penggunaan
```

## ✨ Fitur Utama
1.  **Desain Premium**: Menggunakan palet warna *Deep Dark* dengan aksen *Gold* dan tipografi modern (*Playfair Display* & *Outfit*).
2.  **Animasi Halus**: Menggunakan `framer-motion` untuk efek *fade-in* dan *hover* yang memberikan kesan mewah.
3.  **Responsive**: Dioptimalkan untuk semua ukuran layar (Mobile-friendly).
4.  **Vercel Optimized**: Konfigurasi `vercel.json` sudah tersedia untuk penanganan SPA routing otomatis.

## 🚀 Cara Menggunakan
1.  **Ganti Gambar**: Masukkan foto-foto kelas Anda ke folder `public/assets/`.
2.  **Edit Konten**: Buka `src/App.jsx` untuk mengganti nama kelas, jumlah siswa, dan deskripsi.
3.  **Deploy**: 
    - Push folder ini ke GitHub.
    - Sambungkan ke Vercel.
    - Selesai! Website langsung online.
